# DLE Real Series PRO v7.0

DataLife Engine için geliştirilmiş en kapsamlı dizi/film yönetim sistemidir. 

### Öne Çıkan Özellikler:
- **Tam Hiyerarşi:** Dizi > Sezon > Bölüm yapısını DLE çekirdeğine entegre eder.
- **TMDB Otomasyonu:** Tek tıkla tüm sezon ve bölüm verilerini (oyuncular, fragmanlar, resimler) otomatik çeker.
- **Otonom Cron Motoru:** "Returning Series" durumundaki dizilerin yeni bölümlerini otomatik takip eder ve ekler.
- **Gelişmiş Resim Motoru:** Resimleri sunucuya indirir, watermark basar ve optimize eder.
- **Tasarım Özgürlüğü:** Hiçbir CSS framework dayatmaz; tamamen TPL dosyaları üzerinden kendi CSS'inizle şekillendirebilirsiniz.

### Kurulum:
1. XML dosyasını DLE Plugin Manager üzerinden yükleyin.
2. Admin panelinden TMDB API anahtarınızı girin.
3. TPL dosyalarınızı (series_details.tpl, season_list.tpl vb.) temanıza göre düzenleyin.
